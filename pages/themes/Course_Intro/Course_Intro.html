Change default port of HTTP Preview Server

  1. Go into your workspace directory
  2. open the file .metadata.plugins\org.eclipse.wst.server.core\servers.xml for edit
  3. Find the <server> element for the Preview server in question
  4. Add a port attribute, e.g. port="8181"
  5. Restart Eclipse to pickup the change.
